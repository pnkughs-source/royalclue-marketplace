# Start Here - Local Server

This ZIP starts from your working `royalclue_catalog_minus_hide_toggle_updated.zip`.

## Frontend only

Double click:

```text
START_FRONTEND_LOCAL.bat
```

Then open:

```text
http://127.0.0.1:5500/index.html
```

Admin:

```text
type 2026 on the page
password 2026
```

Direct admin:

```text
http://127.0.0.1:5500/index.html#2026
```

## Backend local

Double click:

```text
START_BACKEND_LOCAL.bat
```

Backend API:

```text
http://localhost:5000/api
```

## Important

For frontend-only testing, admin dashboard, catalog, login, products, support bot, and localStorage features should work without backend.

Backend is only needed for MongoDB/API production-style tracking.
